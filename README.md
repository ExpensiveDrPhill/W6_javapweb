# W6_javapweb

Three-page JavaScript practice site:

- `index.html` — landing page with links
- `pages/registration.html` — registration form with validation
- `pages/dropdown.html` — product dropdown demo and `kodepos` lookup

Shared assets:

- `style.css` — one shared stylesheet for all pages
- `js/app.js` — one shared JavaScript file for validation and dropdown logic

Note: the old `js/kodepos.java` file contains browser JavaScript, but the browser should load a `.js` file instead. The active implementation uses `js/app.js`.
